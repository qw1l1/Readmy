### 1. Цель тестирования
```
  - Проверить корректность работы классов и методов Restaurant:
     
  - Добавление и удаление еды в чеке;
     
  - Проверка суммы заказа;
     
  - Корректный запуск приложения.
```

### 2. Объекты тестирования
  | Компонент | Результат |
  |:------:|:------:|
  | Main | Главный класс системы заказов ресторана |
  | Menu | Класс с меню еды и напитков |
  | MenuItem | Класс с хранением информации о позиций в меню |
  | CafeOrder | Класс с реализацией системы заказов|

### 3. Типы тестов
  | Тип теста | Цель |
  |:------:|:------:|
  | Модульное тестирование | Проверка отдельных методов классов |
  | Интеграционное тестирование | Проверка взаимодействия между классами |
  | Функциональное тестирование | Проверка соответствия программе требованиям |
  | Нагрузочное тестирование | Проверка устойчивости при большом количестве еды в заказе |

### 4. Тестовые сценарии
| Описание | Входные данные | Ожидаемый результат |
|:------:|:------:|:------:|
| Проверить корректное создание объекта MenuItem с валидными данными | name="Пицца", price=12.99, category="Основные блюда" | Создан объект MenuItem с полями: name="Пицца", price=12.99, category="Основные блюда" |
| Проверить обработку некорректной цены при создании MenuItem | name="Чай", price=-5.0, category="Напитки" | Выбрасывается IllegalArgumentException или аналогичное исключение |
| Проверить добавление новой позиции в меню ресторана | MenuItem("Салат Цезарь", 8.99, "Салаты") | Позиция добавляется в меню, доступна для поиска по имени "Салат Цезарь" |
| Проверить поиск существующей позиции в меню по названию | menu.getItem("Паста") при наличии позиции "Паста" | Возвращается объект MenuItem с name="Паста" |
| Проверить инициализацию пустого заказа | new CafeOrder() | Заказ создан, getItems() возвращает пустой список, getTotalAmount() возвращает 0.0 |
| Проверить сброс заказа к начальному состоянию | order.clearOrder() для заказа с 3 позициями общей суммой 25.48 | getItems() возвращает пустой список, getTotalAmount() возвращает 0.0 |
| Проверить корректный запуск приложения | Вызов метода Main.main() | Приложение запускается без ошибок, инициализируются все компоненты системы |
