# Документация проекта Restaurant
Структура:
  | Файл | Назначение |
  |:------:|:------:|
  | test_plan.md | План тестирования. |
  | test_report.md | Отчёт о тестировании. |
  | feedback_summary.md | Анализ отзывов пользователей. |
  | issues.md | Список найденных ошибок и предложений. |

Назначение папки testing:
```
  Папка testing содержит всю документацию проекта Restaurant, включая технические, тестовые и аналитические материалы.
  Эти документы используются для контроля качества, анализа отзывов пользователей и планирования улучшений системы.
```
