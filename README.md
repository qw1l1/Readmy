# Restaurant
- это программа на Java, реализующая рабочий магазин кафе.

### Для кого предназначен:
  - Студенты, которым нужено рабочее кафе
  - Преподаватели для демонстрации работы кафе
  - Разработчики, нуждающиеся в методе работы кафе

### Основные функции:
  - Управление заказами
  - Рабочая система добавления заказа
  - Система удаления товара из заказа
  - Система просмотра заказа
  - Оформление заказа

### Установка и запуск:
  1. Скачать IntelliJ IDEA
  2. Открыть проект в IntelliJ IDEA
  3. Импортируйте все файлы .java
  4. Запустите класс Main

### Примеры использования:
```
  Для запуска программы:
  public static void main(String[] args) {
  Main system = new Main();
  system.start();
  }
```
